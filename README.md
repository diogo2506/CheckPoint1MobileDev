## 📋 Sobre o Projeto

Aplicativo desenvolvido em **React Native** como entrega do **Checkpoint 01** da disciplina de Mobile Application Development da FIAP.

O app consiste em um **formulário de cadastro simples** onde o usuário preenche seus dados e, ao clicar no botão de envio, visualiza as informações inseridas organizadas na própria tela.

---

## ✅ Requisitos Implementados

- [x] `SafeAreaView` como container principal
- [x] `View` para organização dos elementos
- [x] `Text` para títulos, rótulos e informações
- [x] `TextInput` com placeholder em todos os campos
- [x] Campos: **Nome**, **Curso**, **Disciplina** e **Descrição**
- [x] `useState` para controle de cada campo do formulário
- [x] `useEffect` executado na inicialização do app
- [x] `Button` para envio do formulário
- [x] Exibição dos dados preenchidos após o envio
- [x] Estilização com `StyleSheet`
